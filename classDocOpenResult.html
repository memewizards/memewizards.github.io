Detailed Description

Object returned by KdenliveDoc::Open(), containing a pointer to a KdenliveDoc (if successful) and also additional information about whether the doc was modified or upgraded, and any error message. If the doc is nullptr, then errorMessage() will return an error string that can be shown to the user.
