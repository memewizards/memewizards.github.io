Detailed Description

This class represents a Composition object, as viewed by the backend. In general, the Gui associated with it will send modification queries (such as resize or move), and this class authorize them or not depending on the validity of the modifications.

Performs a resize of the given composition. Returns true if the operation succeeded, and otherwise nothing is modified This method is protected because it shouldn't be called directly. Call the function in the timeline instead. If a snap point is within reach, the operation will be coerced to use it.

Parameters
sizeis the new size of the composition
rightis true if we change the right side of the composition, false otherwise
undoLambda function containing the current undo stack. Will be updated with current operation
redoLambda function containing the current redo queue. Will be updated with current operation
