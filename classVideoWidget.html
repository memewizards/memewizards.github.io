Detailed Description

QQuickView that renders an .

Creates an MLT consumer and renders a GL view from the consumer. This pipeline is one of:

A. YUV gl texture w/o GPU filter acceleration B. YUV gl texture multithreaded w/o GPU filter acceleration C. RGB gl texture multithreaded w/ GPU filter acceleration and no sync D. RGB gl texture multithreaded w/ GPU filter acceleration and sync
