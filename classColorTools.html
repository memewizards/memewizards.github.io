Basic HSV plane with two components varying on the x and y axis. If both components are the same, then the y axis will be considered. MIN/MAX give the minimum/maximum saturation, usually 0..255. Missing colour components will be taken from baseColor. For shear == true, the image will be sheared such that the x axis goes through (0,0) and (1,1). offsetY can additionally shift the whole x axis vertically.

Draws a RGB plane with two values on one axis and one on the other. This is e.g. useful as background for a curves dialog. On the line from bottom left to top right are neutral colors. The colors on the y axis show what the neutral color will look like when modifying the curve. color defines the color to modify on the y axis. The other two components will be increased in equal terms (linear as well) on the x axis. scaling \in ]0,1] defines the maximum variance of the selected component; Choosing a value lower than 1 simulates the case that the curves can adjust only +- scaling*255. This mainly delivers a more constant look when also using the Luma component for the curves display but might not represent the actual color change!
