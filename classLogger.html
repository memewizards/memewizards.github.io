Detailed Description

This class is meant to provide an easy way to reproduce bugs involving the model. The idea is to log any modifier function involving a model class, and trace the parameters that were passed, to be able to generate a test-case producing the same behaviour. Note that many modifier functions of the models are nested. We are only interested in the top-most call, and we must ignore bottom calls.
